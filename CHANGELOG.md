# Changelog

All notable changes to CodexGUI will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2026-01-15

### Added
- Initial release as v0.0.1
- GUI wrapper for OpenAI Codex CLI
- Task template system with 10 pre-configured templates
- Real-time streaming output display
- Settings dialog for custom Codex CLI path configuration
- Cross-platform support (Windows, macOS, Linux)
- Full automation mode with `--yolo` flag
- Git repository check skip functionality
- UTF-8 encoding support with Windows cp932 error handling

### Motivation
- Created after realizing CLI intimidates many users (inspired by Claude Code Work articles)
- Designed to make Codex CLI accessible to everyone regardless of command-line experience

### Templates Included
- File Organization
- Business Card Data Extraction
- Code Analysis
- Receipt to Expense Report
- Document Summary
- Photo Organization
- Code Review
- Data Cleanup
- Meeting Notes Summary
- Custom Task
